Hi! A couple of things to note:

  • This copy runs without the sever-side solver. As such you can use Set Parameters to change the solver's priorities or change a debate's importance, but the allocation loaded by Load Allocation is pre-generated and will not change.
  • Similarly, without the server-side component the representational metrics for a panel will not change from zero.
